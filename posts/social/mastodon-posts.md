## Post 2: MRA

Runtime is proof of ignorance.

Every null check, every bounds check, every dynamic dispatch call — each one is a theorem that the type system failed to prove at compile time.

The Minimal Runtime Axiom formalizes this: the optimal runtime is the minimal set of decisions that genuinely require runtime information. Everything else is overhead.
